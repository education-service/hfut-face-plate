LPR是一个分布式车牌识别系统，主要实现技术包括：人工智能、机器视觉和神经网络。

使用方法 : java -jar LPR.jar [-options]

options包括:

-help
提示帮助信息

-gui
运行可视化界面 (默认)

-recognize -i <snapshot>
识别单个图片

-recognize -i <snapshot> -o <dstdir>
识别单个图片，并保存报告信息到指定目录

-newconfig -o <file>
生成默认的配置文件

-newnetwork -o <file>
根据指定的特征提取方法和学习参数训练神经网络（配置文为config.xml），并保存到输出文件中

-newalphabet -i <srcdir> -o <dstdir>
标准化目录<srcdir>中的所有图片，并保存到目录<dstdir>中
